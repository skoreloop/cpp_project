# Симуляция газа Ван-дер-Ваальса

Проект по моделированию реального газа с использованием уравнения состояния Ван-дер-Ваальса.  
Позволяет визуализировать зависимость давления от объёма и температуры, а также наблюдать фазовый переход (конденсацию) при критических условиях.

## Мотивация

Уравнение Ван-дер-Ваальса — это приближение, которое учитывает собственный объём молекул и их притяжение, но оно не описывает детальную динамику частиц.  
Мы пошли дальше: в рамках проекта реализовано **численное моделирование движения молекул** под действием **потенциала Леннарда-Джонса** — более точной модели межмолекулярного взаимодействия.

Зачем это нужно?  
- Сравнить, насколько хорошо уравнение Ван-дер-Ваальса аппроксимирует поведение реального газа.  
- Оценить **ошибку** модели Ван-дер-Ваальса по отношению к молекулярно-динамическому расчёту.  
- Визуально увидеть, когда приближение перестаёт работать (например, при высоких давлениях или низких температурах).

Таким образом, наш проект — это не просто график изотерм, а **верификация** известного уравнения через прямое моделирование взаимодействий частиц.
